Finite Automation

This class represents a finite automaton, which can be either deterministic (DFA) or non-deterministic (NFA).

1. states: A set of states in the automaton.
2. alphabet: A set of input symbols.
3. transitionFunction: An object mapping state-symbol pairs to one or 4. more next states.
5. startState: The initial state.
6. acceptStates: A set of accept (final) states.
